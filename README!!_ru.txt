В репозитории есть файл "info_storage.json". Он содержит настройку скрипта AdsToGS и выглядит как пример ниже, когда не используется:
{
	"credence" : "",
	"ads" :	[
			{
			"access_token":"",
			"ad_account_id": "",
			"sp_sheet_id": "",
			"columns_list": 	[
						],
			"letter_list": 		[
						]
			}
			]
}
Вам нужно заполнить значение "credence" именем json-файла вашего листа Google API-KEY, например "project-aa-abc1234567890.json"

Для добавления рекламного кабинета FB в работу необходимо:
Откройте «info-storage.json» с помощью блокнота или блокнота и добавьте данные в «ads»:
		[
			{
			Три пары «ключ»: «значение»
			и настройки таблицы
			},
			{
			"access_token":"EAAIxXxxXXXXXXXXXXxxxxXXxxxXXXXXXXXXXXXXxxxxxxxxxxxxxxxXX",
			"ad_account_id":"act_12345678901234",
			"sp_sheet_id":"ABCabc1234567890",
			"columns_list":		[
						"date_start",
              					"campaign_name",
             					"ad_name",
         					"spend",
               					"clicks",
               					"cpc"
						],
			"letter_list": 		[
						"A",
						"B",
						"C",
						"D",
						"E",
						"F"
						]
			}
			Вы можете добавить до 50 аккаунтов
		]
Вы можете писать настройки в любых последовательностях 'access_token', 'ad_account_id, 'sp_sheet_id', "columns_list", "letter_list"
Список букв должен соответствовать списку столбцов по индексам, например: "date_start":"A", "campaign_name":"B" ... , как вы хотите видеть его на своем листе
Где взять эти параметры:

ad_account_id — идентификатор рекламного аккаунта FB. Пишется в ads_manager, справа от имени аккаунта в скобках.

sh_sheet_id — идентификатор листа Google. Записывается в адресной строке листа после "https://docs.google.com/spreadsheets/d/" и перед "/edit#gid"

access_token - чтобы получить этот параметр, вам нужно "партнерить" приложение FB по fb_app_id и выполнить 3 шага: Войдите в свою учетную запись разработчика;
На странице приложений выберите приложение, чтобы открыть панель управления для этого приложения; На панели инструментов выберите «Настройки» > «Дополнительно» > «Безопасность» > «Токен клиента».

columns_list — выберите из API facebook_business, используя синтаксис, который Facebook использовал сам

letter_list - выбрать букву для всех столбцов в columns_list в той последовательности, в которой вам нужно записать ее на лист.
Каждая буква соответствует названию столбца по индексу в списке, например: "date_start":"A", "campaign_name":"B"...

ver1.2 Скрипт при запуске заносит в таблицу наличные на рекламном счете с ненулевыми расходами.
Если на текущую дату уже есть данные, то удалит старые и запишет обратно все ТЕКУЩИЕ объявления на рекламном счете с ненулевыми тратами.
